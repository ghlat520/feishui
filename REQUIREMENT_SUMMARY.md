## 二、核心需求（真需求）

### 2.1 用户真正想要的（基于梁宁《真需求》理论）

#### ✅ 情绪价值三维度

**1. 焦虑不安需要保障（60%）**
- 用户痛点：对未来不确定性的恐惧
- 产品回应：提供确定性答案，给予安心
- 话术："让我安心"、"给我方向"

**2. 繁忙生活需要愉悦感（25%）**
- 用户痛点：生活枯燥，缺乏娱乐
- 产品回应：有趣的互动体验，情感宣泄
- 话术："有人听我说"、"释放压力"

**3. 抵抗枯燥需要仪式感（15%）**
- 用户痛点：生活缺乏意义感
- 产品回应：特殊时刻的纪念，仪式感设计
- 话术："今天是特别的日子"、"我有好运加持"

### 2.2 真需求公式
```
真需求 = 缓解焦虑（60%） + 情感宣泄（25%） + 娱乐社交（15%）
```

## 四、产品设计原则

### 4.1 核心原则

**1. 巴纳姆效应**
- 答案模棱两可，人人适用
- 示例："你最近可能会遇到一些挑战，但只要你坚持，就一定能够克服"

**2. 积极偏向**
- 多说好话，少说坏话
- 负面信息转化为建议
- 示例：❌ "你最近运势不好" → ✅ "建议你多注意休息，调整状态"

**3. 模糊限制**
- 留有余地，避免绝对化
- 使用词汇："可能"、"有机会"、"建议"

**4. 行动建议**
- 给出具体可执行的建议
- 增强信任感和实用价值
- 示例："建议本周多穿红色，有助于提升运势"

表面：算命 → 预测未来
