## 一、产品概述

### 1.1 产品定位
基于AI的情感陪伴型玄学娱乐产品，通过星座、塔罗牌、八字等传统玄学元素，为用户提供情绪价值和心理安慰。

**核心价值：**
- 确定性安慰（缓解焦虑）
- 情感宣泄（倾诉出口）
- 娱乐希望（生活仪式感）

### 1.2 目标用户
- **年龄：** 18-35岁
- **性别：** 女性为主（70%）
- **特征：** 
  - 感情/事业迷茫期
  - 生活压力大，需要情绪出口
  - 对玄学感兴趣但不迷信
  - 追求生活仪式感

### 1.3 核心场景
```
触发场景：
- 感情受挫（分手、吵架、暗恋）
- 事业迷茫（换工作、被裁员）
- 财务焦虑（投资亏损）
- 人生节点（毕业、结婚、买房）
- 深夜emo（23:00-02:00高峰期）
```

### 2.2 真需求公式

真需求 = 缓解焦虑（60%） + 情感宣泄（25%） + 娱乐社交（15%）

## 四、产品设计原则

### 4.1 核心原则

#### 原则1：巴纳姆效应
- 答案模棱两可，人人适用
- 示例："你最近可能会遇到一些挑战，但只要你坚持，就一定能够克服"
- 避免：过于具体、可能出错的内容

#### 原则2：积极偏向
- 多说好话，少说坏话
- 即使是负面信息，也要转化为建议
- 示例：❌ "你最近运势不好" → ✅ "建议你多注意休息，调整状态"

#### 原则3：模糊限制
- 留有余地，避免绝对化
- 使用词汇："可能"、"有机会"、"建议"
- 避免："一定"、"肯定"、"必须"

#### 原则4：行动建议
- 给出具体可执行的建议
- 增强信任感和实用价值
- 示例："建议本周多穿红色，有助于提升运势"

### 4.2 话术设计

#### 开场话术
```
"亲爱的{星座}，让我看看星星为你带来了什么信息..."
"在你抽牌之前，请深呼吸，专注于你的问题..."
```

#### 结果话术
```
"牌面显示，你正处在一个转折点..."（模糊）
"星星提示，近期可能会有惊喜等着你..."（积极）
"建议你保持开放的心态，好运自然会来..."（行动建议）
```

#### 付费引导话术
```
"想要更深入的解读吗？解锁详细分析..."
"专属顾问为你一对一解答，帮你理清困惑..."
```

## 六、技术架构（概要）

### 6.1 技术栈
- **前端：** Vue 3 + Vant UI（H5）
- **后端：** Node.js + Express
- **数据库：** MongoDB
- **AI引擎：** 智谱AI API（GLM-4）
- **玄学计算：** lunar-javascript开源库

### 6.2 核心模块
```
用户系统 → 认证、会员
星座系统 → 运势生成、查询
塔罗系统 → 抽牌、解牌、AI对话
八字系统 → 计算引擎（lunar）
支付系统 → 微信/支付宝支付
分享系统 → 社交裂变
```
